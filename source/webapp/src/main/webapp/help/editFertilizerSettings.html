Обратите внимание, здесь Вы можете отредактировать системные настройки. Вы не можете удалить удобрения для работ, которые утсновлены системой, но можете отредактировать удобрения, чтобы они не предлагались автоматически при добавлении работы в Ваш план. Например, если Вы используете только органические удобрения, то для минеральных просто снимите галку с опции Выбирать удобрение по умолчанию при добавлении работы в план работ для данного растения, и Вам не придется удалять предлагаемые минеральные удобрения, если система предлагает их по умолчанию для культуры.

Удобрение может иметь галку "Выбирать по умолчанию при добавлении в мой план", но не присутствовать при добавлении - если в шаблоне удобрения были отредактированы.
